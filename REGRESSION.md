# 双账号完整回归

准备账号 A、B，分别登录两台真机或开发者工具与真机。每项操作后交换账号视角复查。

1. 好友：A 搜索 B 的 6 位 ID 并申请；B 接受；双方好友列表与私信入口一致。
2. 屏蔽：A 屏蔽 B；B 不能向 A 私信，A 不再看到 B 的动态；解除后恢复。
3. 状态隐私：A 关闭“向他人展示状态”；B 在动态、资料和聊天均看不到；重新开启后可见。
4. 已读：A 分别开启和关闭已读回执；B 发送消息并检查“未读/已读”展示是否符合设置。
5. 分享：A 将一条动态分享给 B；B 收到卡片、可打开原动态，分享数只增加一次。
6. 邀请：A 从成员管理勾选好友 B；B 接受行程邀请后进入成员列表，重复接受不会重复入组。

每次发布前先运行 `npm run smoke`，再部署版本检查提示中列出的云函数。
